# Sliding Window

## Core Idea
Use a moving window (left/right pointers) to maintain a valid subarray/substring.

## When to Use
- Contiguous subarray/substring
- Constraints like:
  - "no duplicates"
  - "at most k"
  - "longest/shortest"

## Key Variants
- Fixed window
- Dynamic window

## Problems
- Longest Substring Without Repeating Characters ✅
