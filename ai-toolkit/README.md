# AI Toolkit

A collection of lightweight AI automation scripts, prompts, and workflows for development tasks.

Built around the idea of keeping AI agents **simple, reliable, and reusable**.

## Purpose

This repository contains:

- AI agent runners
- reusable prompts
- automation helpers
- workflow notes
- experiments

The goal is not to create a complicated framework, but to provide small tools that can be reused across projects.
