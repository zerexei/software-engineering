# Code Review

Goal:
Review code quality and find issues.

Instructions:

1. Inspect recent changes.
2. Look for:
   - bugs
   - security issues
   - performance problems
   - missing tests
   - bad patterns

3. Suggest fixes.
4. Implement fixes when appropriate.
5. Run tests.

If everything looks good:
create .complete

If blocked:
create .stuck with reason.
