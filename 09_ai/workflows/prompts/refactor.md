# Refactor

## Goal

Improve codebase structure and readability while preserving existing behavior.

## Instructions

1. Identify files or functions that can be refactored for clean code, readability, or performance.
2. Reorganize, simplify, or modularize the code without altering its external behavior.
3. Run the full test suite to ensure no regressions were introduced.
4. If tests fail, revert or fix the changes.

If refactoring is complete and all tests pass:
- create .agent-complete
- exit

If blocked:
- create .agent-stuck with reason
- exit
